## Webshop Development Notes

### Suggested Product Attributes per SKU
- `id` — unique identifier
- `name` — product name / model
- `brand` — manufacturer brand
- `category` — top-level category
- `subcategory` — e.g. "Traditional Espresso", "On Demand Grinder"
- `description` — product description
- `images` — array of image URLs
- `price` — (to be added)
- `sku` — stock keeping unit code
- `tags` — e.g. ["professional", "HoReCa", "PRO series", "XGi"]
- `available` — boolean
- `featured` — boolean
